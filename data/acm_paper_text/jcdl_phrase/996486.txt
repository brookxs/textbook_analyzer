oclc digital-archive demonstration the demonstration shows the functionality of oclc 's new digital-archive , which became publicly available in september 2002 . this system is an implementation of the iso-reference-model for an open archival information system (oais) . this product uses an implementation of the oclc/rlg preservation metadata element set , tailored for web-documents front -- end workflow and harvester components allow users to archive web-content .
