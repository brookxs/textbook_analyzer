1.5 Toward Effective Information Retrieval Systems

In designing an effective information retrieval system, it is important to understand the components of the systems and the methods that can be used with the system. Types of retrieval system design and evaluation include formal modeling, analytical studies, simulations and user tests.

Common basis for information retrieval systems is matching documents in the system and queries efficiently. Documents in the system is prepared at the lexical level, determining the number of words and phrases in a document before it can be matched to the query presented. The linguistic, pragmatic and heuristic techniques are used to determine the significance of a document in relation to the information need presented through the query given by the user.

Performance of the system is consistently measured by improved algorithms and structures created by researchers. The most commonly used algorithm is the relevance feedback that involves direct evaluation of retrieved documents by the user.

Deeper analysis of the natural language of document and query is carried out as well. World Wide Web documents which contain hypertext links of large volume of data introduces data filtering or data mining to the information retrieval world. This information filtering refers to rapid, inexpensive selection of documents to be processed in more detail for retrieval. Finally, the interface through which the user interacts with the system also plays a major factor in any retrieval system.
