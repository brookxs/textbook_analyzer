11.1 Reference versus Surrogate versus Document

There are four distinctive levels of response that an information retrieval system may make to a query:

  1. Document reference number - most easily accomplished in a computer-based retrieval system as this is probably how the system references the document.
  2. Short reference - title and author, or title, author, and source that gives an overview of the set of retrieved documents.
  3. Full text - presentation of the full text of each document when the document is retrieved.
  4. Full document surrogate - the user receives all of the information that is available in the system
