1.1 Abstraction

There are two abstraction principles that describes the information system:

  1. In any information system, the "real world" is represented by a collection of data abstracted from observations of the real world and made available to the system.
  2. A user's information need, whether for production, storage, or retrieval of information, is abstracted into a form that is commensurate with the information system to be used.

Users of information system have information needs that need to be fulfilled. Therefore, the information system can only serve the users in an abstracted level by matching the data in the information system to the query issued by users that describes the user's information need.
