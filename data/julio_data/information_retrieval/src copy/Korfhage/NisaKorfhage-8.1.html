8.1 Binary versus N-ary Measures

There are two steps in transforming an information need into a query:

  1. Formulate questions that correspond to the information need.
  2. Transform the question into a query that is suitable for a given information system.

The effectiveness of a retrieval system is related to the user's satisfaction with the system output. There are binary and n-ary measure that can be used in user's evaluation.

A binary measure is simplest but presents only a coarse judgment for the user. Each document is either accepted or rejected according to the relevance of the document defined by the user. Difficulties in this measure includes:

  • Relevance in itself is ill-defined.
  • The user has three representations - the need, the question, and the query - against which to judge document.

To be precise, relevance of the document relates to how well the document responds to the query that was asked. How well the document responds to the information need is pertinence. Usefulness is a combination of relevance and pertinence that refer documents that are retrieved but not relevant to the information need present but might be relevant to another information need.
