2.1 The Concept of a Document

A document is a stored data record in any form that includes not only formally printed papers and books, but also informal writing such as messages and letters. A larger body of data is a collection of smaller documents.

A query, on the other hand has different interpretations according to various researchers. Some cite the query to be typically shorter that a document to be retrieved and that the query is different from the documents being sought that it should not be considered a document. Some researchers regard queries as document despite its difference from other documents. For example, in the TREC experiments, the queries are presented in the form of topics that have specific formats and characteristics of a document. Thus, a query should be considered to be a document in this literature.

If a query is considered to be a document, the retrieval process used is a process of matching one document to another. If, however, queries are considered to be distinct from documents, the retrieval process becomes one of mapping documents onto queries. Mapping involves transforming the document into a form or structure that can be directly matched to a query.
