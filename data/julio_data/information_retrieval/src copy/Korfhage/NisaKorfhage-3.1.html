3.1 Matching Criteria

Exact match is possible with numerical or business databases where databases are organized into definite fields. These definite fields make sit possible to determine exactly whether the value of a given term matches the value specified by in the query.

Range match is a type of matching that accepts a range of values for each term. Range matches are possible on terms that have a natural order (numerical order or alphabetical order) so that it is meaningful to specify a term be greater than some minimum value or less than some maximum value.

Most text or image databases are not well adapted to exact or range matching. These types of queries posed are not precise. Therefore, a good approximate match to a query is required to measure how well a given document matches a query. The goal is to satisfy information need exactly and user must settle for approximately matching a query.
