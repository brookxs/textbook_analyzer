9.1 Information the User May Find Helpful

The system uses much information that can help the user to formulate a good query. There are several types of information that the user may find helpful:

  • A user may want to know the number of document to be returned by a query to help determine the need to refine a query.
  • If the system uses a stop list, information about the size of stop list may be helpful and the ability to temporarily remove or add on word to the list will help user formulate an efficient query.
  • Thesaurus is also helpful as it helps user understand how the system interprets a query.
  • Help facility to help guide user into formulating a good query for the system.
