4.2 Boolean-Based Matching

In Boolean retrieval system, each query is a logical function of given words, while a document, in the conventional sense, is not so defined. As there is no structural similarity between document and query, retrieval is based on characteristic function defined on the document space.

Mapping in Boolean query matching is by using the characteristic function and it divides the document space into documents that satisfy the query and document that do not. Since Boolean systems operate on the basis of the presence or absence of terms, retrieval based on the measurement depending on frequency cannot be done.
