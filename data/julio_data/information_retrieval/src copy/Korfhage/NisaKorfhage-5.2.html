5.2 Matrix Representations

There is "many-to many" relationship between terms and documents. Three matrices are used to clarify this relationship. A matrix is a rectangular array of cells holding information.

A term-document matrix, A is a matrix whose rows represent vocabulary terms and whose columns represents documents. A term-term matrix, T, is a square matrix whose rows and columns each represent the vocabulary terms. A document-document matrix, D, is a matrix whose rows and columns represent documents.

The idea behind these matrices is to identify and quantify a relationship between the entities represented as rows and those represented as columns.
