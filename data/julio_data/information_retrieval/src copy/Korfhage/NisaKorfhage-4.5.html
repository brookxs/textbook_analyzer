4.5 Missing Terms and Term Relationships

Missing terms is a problem that arises in the vector space model in which a 0 value is used in two ways: to indicate terms that are truly missing and to indicate terms about which there is no information. One way to solve this is to have the value for any term not specifically included in or excluded from the query to be left undefined.

Vector model arises another problem from the relationships among the terms in a document or query. Results for vector model are defined from a linearly independent set of basis vectors. Linearly independent vector is a set of vectors such that no linear combination adds up to the zero vector. Basis vector is set of vectors from which all vectors within a given vector space can be defined. In the document context, this would mean defining a document by means of terms that have no relationships among themselves. Terms that are used to define the vector model is clearly not independent: a digital computer, for example, is much more common than digital canary.
