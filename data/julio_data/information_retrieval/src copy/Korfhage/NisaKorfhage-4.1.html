4.1 Relevance and Similarity Measures

Document space is a set of documents organized in some specific manner for retrieval. In Boolean retrieval systems, retrieval is based on mapping of document space into query space. Each document that is mapped is transformed into a representation compatible with the query to evaluate the query requirements.

If query cannot be included in document space, characteristic function, that is, function having the value 1 on documents relevant to the query and 0 on documents that are not relevant, is used for evaluation.

Measure is an evaluation of each document on the basis of some computable function to match the query given. Relevance of a document to a query is difficult to measure directly. Therefore, most retrieval systems equate relevance with lexical similarity, that is the matching of words relevant to the query and 0 on documents that are not relevant, is used for evaluation.
