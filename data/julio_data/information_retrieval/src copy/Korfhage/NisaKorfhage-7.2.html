7.2 Documents and Documents Clusters

If an individual document is to be used as a reference point, it must be represented in a way that is compatible with the system. It is also possible to define a reference point from a group of related documents. This can be done by defining a document cluster and determine one or more cluster points that represents the cluster.

Clustering techniques are based on statistical techniques or graph theory. Some systems generate clusters that are distinct, that is, each object is in only one cluster; others generate overlapping clusters. Some clusters begin with everything in one cluster, then successively break this into smaller clusters. Other techniques work in the opposite direction.
