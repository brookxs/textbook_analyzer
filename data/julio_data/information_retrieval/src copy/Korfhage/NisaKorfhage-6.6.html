6.6 The Query and Profile as Separate Reference Points

To retrieve documents, user profile can be used to treat it as a separate reference point in addition to the query. If similarity is measured by distance, a document must be judged for retrieval not only on its distance from the query point but also on its distance from the profile point. Judgments are based on these two distances, using query and profile as co-filters.

There are four ways to combine query and profile distances:

  1. Disjunctive model - requires a document be within distance d of either the query or the profile.
  2. Conjunctive model - requires the distances from a document to both the query and the profile within the limit fixed.
  3. Ellipsoidal model - requires the sum of the two distances be within a prescribed limit.
  4. Cassini oval model - requires the product of the distances from document and query be within a prescribed limit.
