8.5 Operating Curves and Single Measures

Suppose there is some measurable character C, associated with each document in a document collection that is divided into two populations, P1 (relevant documents) and P2 (nonrelevant documents). The values of C can be used to derive an operating curve to a given information retrieval system by plotting the points <%P1, %P2> corresponding to the values of C.

The shape of the curve will depend on the relationship of C to relevance. If high values of C are correlated strongly with relevance, the curve will rise sharply toward high values of %P1. If the documents in P1 and P2 each form a normal distribution with respect to C, then the operating curve is a straight line whose slope depends on the relationship of the characteristic to relevance.

Swets' E measure is determined by measuring the distance of a point from the operating curve for measuring the system effectiveness.
