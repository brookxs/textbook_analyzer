1.5.1 Design

There are many different aspects to IRS design. One aspect is the representation and organization of information in the database. It is important that the user understand the information structure. The user interface is also an important aspect. In some cases the user interface is used to eliminate the need for the user to know the information structure. Query management programs are used to interpret what the user needs. Data management programs are important to system functioning. Presentation to the user is also an important aspect.
