2.4 Knowledge Base

A knowledge base is different from a database. A database is used as a retrieval system, whereas a knowledge base is used to help someone perform a specific function. Knowledge bases can be used to make decisions. The stored information and information inputted into the system by the user are both used as knowledge by the system. Knowledge bases can be in any form. Some common forms are frames and rules. A frame is set of information that describes an entity. A rule is a conditional statement that determines the importance of a piece of information. Knowledge engineers build knowledge bases. These people are usually experts in their filed.
