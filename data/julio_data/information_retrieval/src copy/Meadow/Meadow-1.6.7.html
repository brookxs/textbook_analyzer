1.6.7 The World Wide Web

The World Wide Web (WWW) uses a hypertext model to link documents. Documents can include links to other documents, which can expand a concept. This model is very similar to the human thought process. It is an effective IR model. The Internet is a network originally created to allow scholars to exchange documents. It is based on packet switch network as apposed to a circuit switch network. Tim Berners-Lee created the WWW by using the hypertext model. It has grown astronomically since 1991. It is now a tool used by billions.
