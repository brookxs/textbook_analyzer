5.1 The Concept of Models of Data

A data structure is defined as a collection of data elements and the relationships between the different elements. The model of data is a way to visualize the data structure. The idea of a virtual image in optics and virtual memory in computer science are both discussed. This idea is expanded to data models. The virtual model is what the user sees. If the user is able to interact directly with the virtual data model, then there is data independence.
