4.2.3 Class Membership - Binary, Probabilistic, or Fuzzy

In binary membership, an entity can either by part of the set or not part of the set. There are only two options. Bibliographic classification and indexing is usually done with binary membership. Fuzzy or probabilistic membership is a little more interesting. In this type of classification, it is possible to give weights to different classes. For example, the user could ask for all books about lions with a probability greater than 0.6. This gives more flexibility in classification.
