7.2 Language Types

There are two types of query languages. They are procedural and nonprocedural. A procedural language is a set of commands, such as select, print, and logoff. A nonprocedural language is used to tell the IRS what is wanted. The IRS determines how to get that information. Many users like graphic query representation systems. This type of system shows the query graphically. Menu-based languages give the user lists of options. The return set is limited by what the user chooses.
