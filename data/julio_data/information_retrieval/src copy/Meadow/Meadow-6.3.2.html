6.3.2 Finding Records

There are four ways to find records. They are sequential search, index search, pointer search, and hash search. The sequential search starts the beginning of the file and examines each record until a match is found. An index search uses another structure called an index to determine where the record is generally located. The pointer search follows a linked list until the record is found. A hash search uses the search key to go directly to the location of the record. These search methods are implemented in five different organizational methods. They are sequential files, indexed files, lists, tree structures, and direct access structures.
