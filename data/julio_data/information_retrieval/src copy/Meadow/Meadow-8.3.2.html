8.3.2 Display, Format, and Sort

The format is the way each record should be displayed on the screen. The page size, orientation of the page, selection of attributes to be displayed, and placement of fields and labels are all details that need to be defined. The display mode can be scrolling or record-by-record. Records can be sorted in most systems. In some systems, the user is able to specify the display, format, and sorting options.
